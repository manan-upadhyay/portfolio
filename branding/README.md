# Branding (source & archive)

Source and historical brand assets. **Not deployed** — these live outside `public/`
so they don't ship to production. Production-ready icons/logos live in `public/`
(see [tools/og-image/README.md](../tools/og-image/README.md) for how the icon set
and og-image are derived).

- `source/branding.png` — master brand artwork (concept sheet / high-res source).
- `archive/v1/` — the previous (v1) favicon set, logos, and og-image, kept for reference.
